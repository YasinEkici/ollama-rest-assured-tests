# Yazılım Test Mühendisliği Proje Ödevi:

Rest Assured kütüphanesini kullanarak Java/Maven/jUnit olarak bir servis otomatik
regresyon testi projesi hazırlanması ve sunulması.

Kendi geliştirdiğiniz bir servis için veya online örnek servislerden birine yazılmış olabilir.

En az bir adet çağrım/istek yapılması ve cevap içerisinde kontrollerin sağlanması:

- status code kontrolü 
- response body içerisinde beklenen değer kontrolleri 
- x süre altında cevap dönüldüğünün kontrolü

İdealde GET ve POST kullanarak en az birer örnek yazılması ve bunlardan en az birinde
request body olması. (JSON veya XML request response fark etmez)

Projenin kod geliştirme ve otomatik regresyon testi yazma pratiklerine göre olduğunca güzel
hazırlanması.

- Yazılım Test Mühendisliği (yapay zeka destekli yazılım test mühendisliği) hakkında
    sunum hazırlanması ve sunulması
- Kodunu çalıştırıp testi nasıl yazdığını gösterip sunması
- Kodu kendi public github repoistorylerinize gönderip repository URL bilgisini
    paylaşmanız gerekmektedir
- Sunum dosyanızı da reponuza yüklemeniz gerekmektedir.
